## Сценарии

- блок “Начало сценария” - Блок является стартовой точкой сценария. И определяет передаваемые в сценарий параметры со стороны ВАТС;
- блок “Окончание звонка” - Блок предназначен для завершения разговора по различным сценариям;
- блок “HTTP-запрос” - Блок предназначен для выполнения HTTP-запросов во время исполнения сценария;
- блок “Рандом” - Блок предназначен для случайного направления звонящего по разным веткам сценария;
- добавление/удаление/изменение/перемещение готовых блоков на экране со сценарием;
- устанавливать логические связи между блоками, которые определяют ход выполнения сценария;
- В интерфейсе должна быть возможность вносить номера телефонов для приема входящих обращений и осуществления исходящих звонков. В случае входящих звонков обеспечить возможность настройки маршрутизации номера телефона на определенный сценарий посредством визуального интерфейса.
- Подсистема осуществления обзвонов по расписанию предназначена для осуществления исходящих звонков по определенному интерактивному сценарию. Для создания обзвона необходимо предусмотреть 2 варианта их создания: через графический интерфейс и через вызов HTTP методов.
