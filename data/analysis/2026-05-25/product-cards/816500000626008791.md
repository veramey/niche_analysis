## Сценарии

- Предоставление права использования ранее установленного у Заказчика программного обеспечения: Loginom Server Standard (включая 5 конкурентных лицензий для проектирования сценариев или просмотра отчетов).
- Графический пользовательский интерфейс должен позволять разрабатывать аналитические процессы с помощью визуального проектирования сценариев обработки (технологии low-code).
- аналитический сервер для расчетов, проектирования сценариев, визуализации, управления правами;
- клиентское веб-приложение – рабочее место аналитика для разработки сценариев анализа;
- Возможность пакетного выполнения сценариев по расписанию с использованием встроенного планировщик задач.
- доступ к построению и редактированию сценариев либо только просмотру отчетов;
- Программное обеспечение должно иметь диспетчер для просмотра открытых сессий и пакетов, мониторинга и управления активностью сценариев обработки данных, очистки пула пакетов.
